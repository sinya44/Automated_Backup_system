The Automated Backup & Recovery Management System was developed by Sinya as Python Development Task 02 for the TEYZIX CORE Internship Program. This project was created using Python, Tkinter, SQLite3, Shutil, and Datetime to demonstrate the automation of data backup and recovery processes through a graphical user interface (GUI).

Tkinter was used to design the GUI, enabling users to perform backup and recovery operations easily. The Shutil library was used for file and folder copying, while the Datetime module was used to generate timestamps and maintain backup records. SQLite3 was integrated as the database to store backup information, including source path, destination path, backup date, and backup status.

The system allows users to create backups of important files and folders, maintain backup history in the database, and restore previously saved data when needed.
